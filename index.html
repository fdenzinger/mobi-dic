    Multi-purpOse displacement Behavior Identifier Digital Image Correlation Tool

    Sub-pixel displacement tracking for georeferenced and non-georeferenced raster data, developed at the WSL Institute for Snow and Avalanche Research SLF (Davos) by the Alpine Remote Sensing Group.

    What MOBI-DIC can do¶

    • Native QGIS plugin — ready in minutes

      Install once and run directly inside QGIS. The tabbed dialog guides you through every step — configuring single-pair or multi-date time series inputs, tuning DIC parameters, filtering vectors, and loading results with automatic symbology. Run configurations can be saved, reloaded, and shared. Displacement outputs land directly in the QGIS layer tree for immediate further analysis.

    • 3D surface displacement from DEMs

      Derives East-West, North-South, and vertical displacement from repeat DEMs using hillshade or orientation image correlation. The secondary DEM is then shifted back by the measured horizontal offsets, so the vertical component (DZr) reflects true elevation change without horizontal motion bias.

      Output: HDF5 + GeoTIFF per component.

    • 2D displacement from optical imagery

      Tracks surface motion in any georeferenced image pair: orthophotos, satellite scenes, or UAV mosaics. Automatic co-registration removes sensor offsets before correlation for cleaner results.

      Output: HDF5 + GeoTIFF (EW, NS).

    • Image-based displacement tracking

      Processes non-georeferenced image pairs from fixed cameras, drones, or scanners. Optional AI segmentation masks sky, water, and other non-moving regions before correlation.

      Output: HDF5 + displacement images (pixel units).

    • Post-processing & outlier filtering

      Two independent filters — neighbourhood coherence and iterative neighbour similarity — remove outliers without re-running DIC. Thresholds can be tuned interactively and filtered results exported as GeoPackage or PNG.

    • Google Earth export & 3D visualisation

      Export displacement rasters and vector arrow fields as KMZ directly from the plugin. Results open in Google Earth Pro for interactive 3D inspection, with optional TimeSpan tags for animated time-series playback.

    • Print layout builder

      Create publication-ready displacement maps with one click. Choose from dedicated displacement colormaps, overlay hillshade or satellite basemaps, and open the result directly in QGIS's print layout editor for PDF or PNG export.

    • CLI & headless batch processing

      Drive the full pipeline from YAML config files — on a desktop, a server, or inside an automated workflow. Sweep multiple resolutions and window sizes in a single run, with all outputs written automatically. No GUI required.
